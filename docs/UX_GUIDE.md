# UX Guide — Cómo usar Personal Console

## 1) Entrada (captura y triage)
- Usá **Entrada** para capturar tareas rápidas.
- Toda tarea nueva entra como `backlog` en `P4` si no tiene fecha.
- Desde cada tarjeta podés asignar carril, poner fecha y estado.

## 2) Tablero (prioridad temporal)
- El tablero muestra solo carriles: **P0 Hoy, P1 Semana, P2 Mes, P3 60 días, P4 Algún día**.
- El estado (backlog/en curso/bloqueada/hecha/archivada) vive dentro de la tarjeta.
- Las tarjetas se ordenan por riesgo y muestran motivos para ayudarte a decidir.

## 3) Enfoque (foco de hoy)
- Elegí una tarea y trabajá en una sola cosa.
- Si no sabés cuál elegir, usá **Sugerir foco**.
- Vas a ver riesgo, motivos y acciones rápidas: en curso, bloquear, hecha, mover carril y cambiar fecha.

## 4) Calendario
- Una sola pestaña con selector **Mes/Semana**.
- Mostramos tareas con fecha de vencimiento.
- Podés arrastrar para reprogramar.

## 5) Settings y sincronización
- Si cargás la URL del Web App, sincronizás con Google Sheets.
- Si no cargás URL, la app sigue funcionando offline en este navegador.
- Botones:
  - **Probar conexión**: valida endpoint.
  - **Enviar tarea de prueba**: verifica escritura.
  - **Sincronizar ahora**: fuerza push/pull.
